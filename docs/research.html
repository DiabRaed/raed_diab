1. Optical Alignment for High-Precision Systems

This project explores advanced techniques for aligning optical components with micrometer-level precision. The work focuses on the integration of laser interferometry and image processing to automate alignment tasks, reducing human error and improving reproducibility. This is part of the LIGO group.

2. Automation of Optical Bench Calibration

A key challenge in optical instrumentation is consistent calibration over time. This project involved building a robotic platform that can measure, analyze, and calibrate optical benches using feedback loops and AI-based optimization. It supports integration into both research and industrial settings.
